# CSS-2: GEN AI Card Layout

## Description
A responsive card layout built with HTML and CSS, demonstrating the use of **viewport units** (`vw` and `vh`) for spacing and sizing. The card features a header, an image section, and a styled button with a gradient background.

## Features
- **Viewport-based spacing**: `25vw` left/right margin and `10vh` top/bottom margin
- **No scrolling**: The entire layout fits within the viewport
- **Gradient button**: Styled with a linear gradient background and custom font
- **Responsive sizing**: Card dimensions use `calc()` with viewport units

## How to Run
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Open `index.html` in any modern web browser.
3. Alternatively, use a Live Server extension in VS Code for auto-reload.

## Technologies Used
- HTML5
- CSS3 (Viewport Units, Flexbox, Linear Gradients)
